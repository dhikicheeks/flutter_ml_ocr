
# flutter_ml_ocr

Flutter ML OCR Example.
_Machine Learing_
_Optical Character Recognition_

## Getting Started

This project covers implementation of text recognition in flutter using Google ML kit. Flutter_ml_ocr is implemented by selecting image from camera.
